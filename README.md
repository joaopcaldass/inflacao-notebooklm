# Inflação — Estudo Introdutório com IA Generativa
Este repositório apresenta um projeto educacional desenvolvido com apoio do NotebookLM, utilizando IA generativa como ferramenta de aprendizagem ativa para organizar, analisar e sintetizar conteúdos sobre inflação a partir de fontes abertas.

## Objetivo
Desenvolver um caderno temático a partir de fontes abertas, explorando o uso da IA na formulação de perguntas estratégicas, testes de prompts e organização do conhecimento.

## Metodologia
- Curadoria de fontes abertas (textos e PDFs)
- Definição de objetivos de estudo
- Elaboração de perguntas estratégicas
- Testes de variações de prompts no NotebookLM
- Registro das respostas com referências
- Síntese dos conteúdos em materiais reutilizáveis

A IA foi utilizada como apoio ao aprendizado, mantendo análise crítica humana em todas as etapas.

## Conteúdo do Repositório
- **mini-guia.md** — Síntese estruturada sobre inflação  
- **glossario.md** — Conceitos fundamentais  
- **perguntas-estrategicas.md** — Questões orientadoras do estudo  
- **prompts-reutilizaveis.md** — Prompts aplicáveis a outros temas  
- **referencias.md** — Fontes utilizadas no projeto  

## Ferramentas Utilizadas
- NotebookLM
- GitHub

## Observações
Este projeto foi desenvolvido com fins educacionais, demonstrando o uso responsável da IA generativa como ferramenta de apoio ao aprendizado, com curadoria de fontes e pensamento crítico humano.
